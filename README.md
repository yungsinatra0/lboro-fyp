## Troubleshooting

### Biber Issues

1. Delete all auxiliary files:
    - .aux
    - .bbl
    - .bcf
    - .blg
    - .run.xml
2. Run `biber --version`
3. If there's an error in step 2, delete all files in the error directory
4. Rerun the biber recipe in the Latex Workshop menu
